# Church Enneagram Test

교회 공동체 안에서 자신을 돌아보고 이해하기 위한  
**에니어그램 간이 테스트 웹 페이지**입니다.

복잡한 이론 설명보다는 일상적인 선택과 반응을 통해 자신의 경향성을 가볍게 확인하는 데 목적이 있습니다.  
그렇기에 본 테스트 결과는 개인의 성향을 단정하거나 신앙의 성숙도, 영성의 깊이를 평가하지 않습니다.  
참고용으로만 활용해 주세요.


---

## ✨ 특징

- **30문항 구성**의 간단한 에니어그램 테스트
- 문항당 **1페이지 흐름**으로 집중도 높은 진행
- 마지막 문항은 **유형 노출 없이 선택**하도록 설계
- 결과는 **상대 점수 기반**으로 표시 (순위 + 그래프)
- 모바일/데스크톱 모두 대응

---

## 🎯 대상

- 교회를 다니는 성도
- 소그룹 / 셀 모임 / 청년부 / 새가족 교육
- 에니어그램을 처음 접하는 분들

> 본 테스트는 **진단이나 분류를 위한 도구가 아니라**,  
> 자기 이해와 대화를 돕기 위한 참고 자료입니다.

---

## 🖥️ 기술 스택

- HTML / CSS / Vanilla JavaScript
- 외부 라이브러리 의존성 없음
- 정적 페이지 구성
